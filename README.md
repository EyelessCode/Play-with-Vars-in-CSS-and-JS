# 🎛️ Play with Vars

Proyecto simple para experimentar con variables de **CSS** controladas dinámicamente mediante **JavaScript**.

Permite modificar propiedades visuales en tiempo real usando controles como sliders y selectores de color.

## 🚀 Características

- 🎚️ Control de **espaciado (spacing)**
- 🌫️ Ajuste de **blur (desenfoque)**
- 🎨 Cambio de **color base**
- ⚡ Actualización en tiempo real con JavaScript
- 🧩 Uso de variables CSS (`--var`)

## 🛠️ Tecnologías

- HTML5
- CSS3 (Variables CSS)
- JavaScript (DOM + eventos)

## 🌿 Flujo de trabajo (Git)

El proyecto se organiza usando múltiples ramas:

- `main` → versión estable
- `develop` → integración de features
- `release` → preparación de versiones
- `feature/page` → estructura HTML
- `feature/style` → estilos CSS
- `feature/script` → lógica JavaScript

## 💡 Aprendizaje

Este proyecto es ideal para entender:

* Cómo funcionan las variables CSS
* Manipulación del DOM con JavaScript
* Eventos (input, change)
* Integración entre estilos y scripts


## 📌 Notas

No requiere instalación ni dependencias.
Funciona directamente en el navegador.
Proyecto creado con fines educativos.
